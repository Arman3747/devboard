Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an element in the DOM. They are:

01 . getElementById()
02 . getElementsByClassName()
03 . getElementsByTagName()
04 . querySelector() =>eg. querySelector('.fruits-class')
05 . querySelectorAll() => 2 types using id and class.
05a . querySelectorAll('#fruit-id')
05b . querySelectorAll('.fruit-class')

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML:
We can get or set the HTML content from the JavaScript object. innerHTML returns us the content and tags both together. For example innerHTML is returning us the full tag: - <div class="font-bold"><p>Hello, World!</p></div>.

innerText:
We can get of set only the visible text content inside a tag. It does not give us the HTML tag. For Example we can only get:- "Hello, World!". And it does not give us the tag.

textContent:
We can get or set the HTML content from the JavaScript. It also gives us the hidden elements. For example: <!-- <div class="font-bold"><p>Hello, World!</p></div> -->

Question-3: What is event delegation in the DOM?

In Event delegation, a parent node handles all the events for the child node. Helps to code less as we can call the parent node and allocate the task to its child node.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event triggered on a child element it goes up to its parent elements which allows the parents to handle the event.
